## Overview

Expense Tracker Dashboard is a web application developed using React.js and Tailwind CSS that helps users track their income and expenses efficiently.

The application provides an intuitive dashboard interface with real-time financial statistics, interactive charts, and transaction management features. It helps users understand their spending habits through visual analytics like pie charts and trend graphs.

---

## Features

* Add Income and Expenses
* Categorize Transactions (Food, Travel, Shopping, etc.)
* Edit and Delete Transactions
* Real-time Balance Calculation
* Monthly Filter for Transactions
* Interactive Pie Chart (Income vs Expense)
* Line Chart for Spending Trends
* Local Storage Data Persistence
* Responsive Dashboard UI
